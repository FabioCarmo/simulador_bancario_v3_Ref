
#simulador de sistema bancario versao: 3.0 Refatorado

Projeto de Refatoração

Esse projeto tem como intuito evoluir um código legado e com erros
para uma estrutura de código limpa e funcional.

Ao implementar Python DBAPI para armazenar os dados na base de dados
a estrutura do código mudou dinamicamente e o sistema se tornou complexo
e com erros, tendo a necessidade de refatorar a estrutura principal como
as interações entre funções e objetos com o DB.
